# Job Portal - SMS.ru авторизация

## Быстрый старт

### 1. Запуск Backend

```bash
cd job-portal-server
npm run start:dev
```

Backend запустится на `http://localhost:5000`

### 2. Запуск Frontend

**Откройте новый терминал:**

```bash
cd job-portal-client
npm run dev
```

Frontend запустится на `http://localhost:5173`

### 3. Тестирование авторизации

1. Откройте в браузере: `http://localhost:5173`
2. Нажмите **"Вход"** (правый верхний угол)
3. Введите телефон: `+79991234567`
4. Нажмите **"Получить код"**
5. **Смотрите консоль backend** - там будет код (например: `4542`)
6. Введите код, имя и выберите роль
7. Нажмите **"Войти"**

## Тестовый режим

Коды НЕ отправляются по SMS, а **логируются в консоль сервера**.

Ищите строку: `Код верификации для +7999...: XXXX`

## Роли пользователей

- **Соискатель** (`job_seeker`) - ищет работу, откликается на вакансии
- **Работодатель** (`employer`) - создаёт вакансии

**Только работодатели могут создавать/редактировать/удалять вакансии!**
